## Getting Started

### Backend Setup (FastAPI)

1. Open a terminal / command prompt.
2. Navigate to the project root folder:
   ```bash
   cd path/to/project/root
   ```

3. Move to the backend folder:
  ```bash
  cd backend
   ```
4. Activate the virtual environment:
   
   Windows
   ```bash
   venv\Scripts\activate
   ```
   Unix/Mac
   ```bash
   source venv/bin/activate
   ```
5. Start the FastAPI server
    ```bash
    uvicorn main:app --reload
   ```

### Frontend Setup (React + Vite)
1. Open a second terminal/command prompt
2. Navigate to the project roo folder
   ```bash
   cd path/to/project/root
   ```
3. Install Frontend dependencies (if not already installed)
   ```bash
   npm install
   ```
4. Start the Frontend dev server
   ```bash
   npm run dev
   ```
   The Frontend runs by default at http://localhost:5173
